# Фикстуры для eval

`cases.json` содержит маленький детерминированный eval-набор для кэпстоун-проекта.

Он проверяет три вещи:

1. вопрос по планированию capacity находит правильные raw-источники;
2. вопрос по инциденту Aurora-17 находит правильный raw-источник;
3. вопрос вне корпуса не должен притворяться ответом с опорой на источники.
