# M0

## Goal

В этом milestone мы делаем репозиторий живым сразу после первого запуска.

Система должна:

- оставаться детерминированной и local-first;
- собрать базовый `vault/` с `raw`, `wiki`, `outputs`, `index.md` и `log.md`;
- сгенерировать один grounded answer;
- записать `run record` и `trace`.

## Scope

Нужно сделать:

- загрузить локальный markdown corpus из `vault/raw/`;
- собрать `vault/wiki/index.md`;
- записать grounded answer в `vault/outputs/`;
- сохранить `run record` и `trace`;
- поддержать `vault/index.md` и `vault/log.md` как стабильные точки входа.

Out of scope:

- live web search
- paid adapters
- embeddings
- write-back
- runtime governance

## Debugging Cues

- если `vault/` не собирается, сначала проверь существование `vault/raw/`
- если answer не появился, проверь, не упал ли run до этапа записи output
- если trace пустой, проверь вызовы trace layer вокруг demo path
