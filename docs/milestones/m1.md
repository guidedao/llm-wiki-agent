# M1

## Goal

В этом milestone wiki перестаёт быть побочным продуктом и становится реальной
query surface.

Система должна:

- ориентироваться через `index -> concepts -> sources -> raw`;
- делать сбор контекста inspectable;
- сохранять context packet как отдельный артефакт.

## Scope

Нужно сделать:

- собрать `vault/wiki/sources/` и `vault/wiki/concepts/`;
- выбирать wiki pages перед обращением к raw notes;
- сохранять `artifacts/context/<run_id>.json`;
- сделать в trace видимыми этапы wiki retrieval и context assembly.

Out of scope:

- planning layer
- embeddings / vector search
- write-back
- runtime health checks

## Debugging Cues

- если `concepts/` не появились, проверь этап compilation before query
- если context packet пустой, проверь resolve path между wiki pages и raw notes
- если answer снова собирается напрямую из raw, проверь порядок retrieval stages
